# Үйлену тойына шақыру / Свадебное приглашение 💍

Бұл — Алтынбек пен Аяужанның үйлену тойына арналған электронды шақыру сайты.

## Мүмкіндіктері / Особенности
- **Шымылдық анимациясы (Welcome Overlay):** Сайт ашылған кездегі әдемі, баяу және элегантты ашылатын шымылдық (экрана басып ашу).
- **Интерактивті дресс-код (Interactive Dress Code):** Қонақтар палитрадағы түстерді таңдап, костюм мен көйлектің түсін өзгертіп көре алады.
- **Кері есептегіш таймер (Countdown Timer):** Той күніне дейін қалған уақытты көрсетеді.
- **Той бағдарламасы (Timeline):** Үйлену тойының өту барысы (Бағдарламасы).
- **RSVP қатысуды растау формасы (RSVP Form):** Қонақтардың тойға келетінін растау формасы.
- **Музыкалық ойнатқыш (Background Music):** Сайтта фондық әуенді қосу/өшіру мүмкіндігі.

## Технологиялық стек / Стек технологий
- HTML5 (Семантикалық құрылым)
- CSS3 (Элегантты дизайн, шторка анимациялары, мобильді бейімділік)
- JavaScript (Таймер логикасы, интерактивті дресс-код, RSVP жіберу, әуен басқару)

## Жоба құрылымы / Структура проекта
- `index.html` — Негізгі беттің коды.
- `style.css` — Стильдер мен дизайндар жинағы.
- `script.js` — Интерактивті элементтер мен функциялар логикасы.
- `couple.jpg` — Жастардың суреті.
- `music.mp3` — Фондық әуен файлы.
- `rings.jpg` / `hands.jpg` — Декоративті суреттер.
